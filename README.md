A sleek single-page profile built with React, Vite, and Tailwind CSS. The site showcases experience, projects, education, and provides quick access to social links.

## Features

- Sticky sidebar with active-section indicator powered by Intersection Observer
- Spotlight cursor effect and smooth scrolling for delightful UX
- Responsive layouts for About, Experience, Projects, and Education
- Accessible navigation, semantic HTML, and keyboard-friendly focus states

## Tech Stack

- React 19 + Vite 7
- Tailwind CSS 3
- Custom hooks for scroll spying

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
npm run preview
```

## Deployment

Deploy the contents of `dist/` to any static host. The project is optimized for platforms such as Vercel, Netlify, or GitHub Pages.
